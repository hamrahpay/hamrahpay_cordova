# Hamrahpay Library for Apache Cordova / PhoneGap

Supported Platforms:

 * Android
 * iOS
 * Windows Phone
 * Tizen
 * Browser
 * BlackBerry
 * Windows
 * OSX
 * Firefox OS
 * Amazon Fire OS




And here's some sample code! :+1:

```javascript
var sku = 'hp_5415e384f37bf802917441'; // Replace it with your sku
$('#pay_btn').click(function(){
		hamrahpay.PayRequest(sku,device.uuid);
});
```
